# CIFAR-10普适物体识别
## 代码环境
- **Windows** 10
- **Python** 3.10
- **PyTorch** 1.11.0
- **Torchvision** 0.12.0
- **CUDA** 11.2
- **Tqdm** 4.64.0
- **Wandb** 0.12.16

## Usage
运行`main.py`.
